# Sidefold Landing Page

Landing page para promocionar la extensión de Chrome **Sidefold** — organiza tus suscripciones de YouTube con carpetas y filtros.

## 🚀 Características

- **Vite + React 18** - Build rápido y moderno
- **Componentes modulares** - Arquitectura limpia y mantenible
- **Responsive design** - Funciona en todos los dispositivos
- **Animations** - Scroll reveal y transiciones suaves
- **Optimizado para SEO** - Meta tags y estructura semántica
- **PWA Ready** - Manifest y service worker

## 📦 Requisitos

- Node.js 16+
- npm o yarn

## 🛠️ Instalación

```bash
npm install
```

## 👨‍💻 Desarrollo

Inicia el servidor de desarrollo en http://localhost:5173/:

```bash
npm run dev
```

## 🔨 Producción

Build optimizado para producción:

```bash
npm run build
```

Los archivos compilados se generan en la carpeta `dist/`

Previsualizalo localmente:

```bash
npm run preview
```
